# NinjaAiBot — ربات همه‌کاره فروش + هوش مصنوعی

## ویژگی‌ها
- فروش PDF, تصویر, NFT, لینک
- پرداخت خودکار USDT روی TON
- تولید متن، خلاصه، تصویر
- تبلیغ خودکار در گروه‌ها
- لینک دعوت وایرال
- پنل ادمین داخل تلگرام

## نحوه استقرار
1. یک ربات در @BotFather بسازید
2. یک پروژه جدید در [Render.com](https://render.com) ایجاد کنید
3. فایل‌ها رو آپلود کنید
4. متغیرهای محیطی زیر رو تنظیم کنید:
   - `BOT_TOKEN`: توکن ربات
   - `ADMIN_ID`: آیدی عددی تلگرامی شما
   - `GROQ_API_KEY`: از [groq.com/console](https://groq.com/console) رایگان دریافت کنید
   - `REPLICATE_API_TOKEN`: از [replicate.com/account](https://replicate.com/account)
